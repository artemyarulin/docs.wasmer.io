## Previous Releases

- [Wasmer 6.0.0](/releases/6.0.0) (April 24, 2025) - Added support for the exception handling proposal on Linux

## Finding Your Wasmer Version

You can check your current Wasmer version by running:

```bash
wasmer --version
```

## Upgrading Wasmer

To upgrade to the latest version:

```bash
# Unix/Linux/macOS
curl https://get.wasmer.io -sSfL | sh

# Windows (PowerShell)
iwr https://win.wasmer.io -useb | iex
```

To install a specific version:

```bash
curl https://get.wasmer.io -sSfL | sh -s "v6.0.1"
```
